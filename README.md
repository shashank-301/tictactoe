# Tic-Tac-Toe

This is a sample repo created by Copilot.

## Features

- Play Tic-Tac-Toe in Python (CLI)
- Web interface using Flask
- Unit tests for core logic

## How to run

### CLI

```
python tictactoe.py
```

### Web App

```
pip install -r requirements.txt
python webapp.py
```

### Run Tests

```
pytest
```
